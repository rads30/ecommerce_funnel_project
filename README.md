## Overview
This project analyzes an e-commerce purchase funnel to identify drop-offs, understand user behavior, and provide actionable recommendations. Designed in a **Corporate Style** for executives, focusing on insights over code.

---

## Problem Statement
Users drop off at multiple stages in the purchase funnel (Page View → Product View → Add to Cart → Checkout → Purchase). Objective: identify largest drop-offs, segment by device and category, and provide prioritized recommendations.

---

## Tools & Skills Used
- Python, Pandas, NumPy, Matplotlib, Seaborn  
- Data visualization & corporate-style charts  
- Funnel & drop-off analysis  
- SQL-style queries (pseudo)  
- Executive insights, A/B testing recommendations, stakeholder reporting

---

## Key Features
- Funnel analysis with drop-off percentages  
- Conversion analysis by **device** & **product category**  
- Segmentation by **New vs Returning users**  
- Executive-ready visualizations (saved in `dashboard/screenshots/`)  
- Actionable recommendations to improve conversion and retention

---

## Insights & Recommendations
1. Largest drop-off: Add-to-Cart → Checkout → simplify checkout flow  
2. Mobile users convert less → prioritize mobile UX optimization  
3. Returning users convert better → focus on retention campaigns  
4. Low-conversion categories → improve content & merchandising  
5. Instrument tracking → ensure reliable data for experiments  
6. Create a self-service funnel dashboard for Product & Marketing
